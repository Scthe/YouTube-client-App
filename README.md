YouTube client App
==================

Test app for some of the most recent/interesting technologies.

## How to run
1. run `npm install`
2. run `bower install`
3. `grunt`

## Used technology

* YouTube API v3
* [underscore]
* [Backbone.js]
* [RequireJS]
* [Bacon.js]
* [jQuery]
* [JSHint]
* [scss]
* oocss
* [Bootstrap]
* [grunt]
* [bower]
